• Define your color map in CSS
    • :root{
    --primary-main: #9fa5fe;
    --primary-light: #c0effe;
    --primary-dark: #4802fe;
}
  • Then you can read them in JS


  • Define css variables for a component
    • .button{
    color: var(--ui-button--color, #eb2f73);
    background-color: var(--ui-button--background-color, #ffffff);
}

  • Theming using plain CSS
    • // DOM Tree: .override-colors .button
.override-colors {
    --ui-button--color: #41eb09;
    --ui-button--background-color: #000000;
}
